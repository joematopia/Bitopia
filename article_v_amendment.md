# 🏛️ ARTICLE V: The Constitutional Amendment Protocol
**Classification:** System Upgrade & Network Evolution
**Security Level:** Maximum (Time-Locked & Supermajority Required)

## 1. The Core Philosophy
A system that cannot bend will eventually break. However, a system that bends too easily will be captured by malicious actors. The Bitopia Codex (The Constitution) is immutable on a daily basis, but can be upgraded via the Article V Protocol. This ensures Bitopia outlives its original Architects.

## 2. Phase I: The Proposal (Proof-of-Skin-in-the-Game)
To prevent network spam and frivolous changes, proposing an Amendment requires a massive financial commitment.
* A Citizen (or coalition of Citizens) must lock **10,000 $GOV** into the `Amendment_Staking_Contract`.
* If the Amendment fails to reach a 20% approval threshold, the staked `$GOV` is slashed (burned) to penalize wasting the Network's time.

## 3. Phase II: The 30-Day Terrestrial Time-Lock (Debate)
Once proposed, the Amendment is broadcast to the Bitaris Swarm and enters a mandatory **30-Day Time-Lock**.
* **Zero-Day Protection:** The code cannot be executed during this time.
* **The Public Square:** Citizens analyze the code, debate the philosophical implications, and audit the smart contract for exploits. 

## 4. Phase III: The Quadratic Supermajority (Ratification)
Bitopia does not operate on mob rule (51%). To alter the foundational Codex, the Amendment must achieve a **75% Supermajority** approval from all active `$GOV` voters.
* The voting is **Quadratic** ($V = \sqrt{S}$), ensuring that a few massive whales cannot force an amendment through against the will of the broader Citizenry.

## 5. Phase IV: Execution (The Bitaris Handshake)
If the 75% threshold is met, the terrestrial nodes sign the transaction and pass it to the MEO CubeSats. The **Bitaris Swarm** implements the code upgrade simultaneously across the orbital network, successfully evolving the Bitopia Network State.
